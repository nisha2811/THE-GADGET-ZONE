# THE-GADGET-ZONE
Online Portable Electronic Devices Store


# OVERVIEW
The Smart Zone is an e-commerce web application designed for buying portable electronic devices such as mobiles, laptops, tablets, smartwatches, headphones, and accessories. It provides a seamless online shopping experience with secure payment options and efficient order management.


# FEATURES

1) Browse and search products
2) Add to cart & manage cart items
3) User authentication (Login/Registration)
4) Multiple payment options: -
   a) Online Payment (Card/UPI)
   b) Cash on Delivery (COD)

5) Order tracking & management
6) Return and complaint handling


# ADMIN DASHBOARD

1) Product management
2) Order tracking
3) Customer insights
4) Manage products and stock
5) View customer activity
6) Track orders and deliveries
7) Update system data


# CUSTOMER

1) Browse products
2) Add to cart and checkout
3) Choose delivery method
4) Track orders and manage account


# TECH STACK

A) Frontend :-

    1) HTML5
    2) CSS
    3) JavaScript (ES6)

B) Backend :-

    1) PHP (v7.4)

C) Database :-
    
    1) PostgreSQL (v12.3)

D) Tools & Platforms :-

    1) Ubuntu / Windows OS
    2) MS Access (for design/reference)

E) Database Tables :-

    1) admin – Admin details
    2) customer – Customer data
    3) product – Product catalog
    4) stock – Inventory management
    5) bill – Order billing
    6) short_bill – Bill-product mapping
    7) temp_cart – Cart items


# OBJECTIVES 

1) Provide secure and reliable shopping
2) Improve customer satisfaction
3) Handle large-scale orders
4) Deliver products on time
5) Maintain data integrity and security
